About the Background

本项目为我在优达学城数据分析纳米学位的毕业项目，目的是设计一个合理的ab测试。在本项目中，我所要考虑的是由优达学城运行过的一个真实试验。具体数字已做更改，但是模式并没有改变。我需要将试验的想法变成一个完整定义的设计、分析其结果，并提供一个高层次的后续实验。

在进行此试验时，优达学城当前的主页上有两个选项：“开始免费试学”和“访问课程资料”。如果学生点击“开始免费试学”，系统将要求他们输入信用卡信息，然后他们将进入付费课程版本的免费试学。14天后，将对他们自动收费，除非他们在此期限结束前取消试用。若学生点击“访问课程材料”，他们将能够观看视频和免费进行小测试，但是他们不会获得导师指导支持或验证证书，无法提交最终项目来获取反馈。

The Task

任务

在此试验中，优达学城测试了一项变化，如果学生点击“开始免费试学”，系统会问他们有多少时间投入到这个课程中。如果学生表示每周5小时或更多，将按常规程序进行登录。如果他们表示一周不到5小时，将出现一条消息说明优达学城的课程通常需要更多的时间投入才能成功完成，并建议学生可免费访问课程资料。在这里，学生可选择继续进行免费试学，或免费访问课程资料。这张截图展示了试验概况。

  • 我们的假设是这会为学生预先设定明确的期望，从而减少因为没有足够的时间而离开免费试学，并因此受挫的学生数量，同时不会在很大程度上减少继续通过免费试学和最终完成课程的学生数量。
  • 如果这个假设最后为真，优达学城将改进整体学生体验和提高导师为能够完成课程的学生提供支持的能力。
  • 转移单位为 cookie，尽管学生参加的是免费试学，但在登录后他们的用户 id 便被跟踪。同一个用户id不能两次参加免费试学。
  • 对于不参加免费试学的用户，他们的用户id不会在试验中被跟踪，即使他们在访问课程概述页面时登录了网站。

Experiment Design

— 实验设计

Metric Choice

选择和描述指标

不变指标

cookie是分组单元，肯定是不变的，测试的是首页的子页面，不会影响首页的情况，也就不会影响cookie数量；首页没有任何改变，cookie不变，点击次数也不变；点击次数不变，cookie不变，点进概率也不变；

  • Number of cookies（Cookie的数量）
  • Number of clicks（点击次数）
  • Click-through-probalility（点进概率）

评估指标

因为提醒用户每周5小时学习时间，可能会影响完成登录并参加免费试学的用户id的数量，是我们想通过A/B测试来研究的问题，实验预期是减少无法一周学习5小时的人离开免费试学，即：分子会减小，分母不变，因此总转化率期望是减小；因为提醒用户每周5小时学习时间，会对付费用户和完成免费试用的用户的比例产生影响，是个很好的评估指标，因为根据预期，期望最终通过免费试学和最终完成课程的学生数量不变，而登录的用户id数量减少，根据公式，留存率会增大；为提醒用户每周5小时学习时间，会对付费用户产生影响，因此净转化率是个很好的评估指标，根据实验预期，付费用户不变，点击“开始免费试学”按钮的唯一cookie数量不变，所以净转化率的期望结果不变

  • Gross conversion（总转化率）期望结果：减小
  • Retention（留存率）期望结果：增大
  • Net conversion（净转化率）期望结果：不变

Measuring Standard Deviation

计算标准偏差

计算标准偏差

对于选择作为评估度量的每个度量，假设有5000个 cookie 样本大小访问课程概述页面的情况下，估计了一下其标准偏差。

  • Std dev = sqrt（0.20625 * （1-0.20625） / 5000*0.08） = 0.0202

    • 总转化率是以cookie数量作为分母，也是转移的单位。转移单位等于分析单位，表明分析估计值与经验变异类似

  • Std dev = sqrt（0.53 * （1-0. 53） / 5000*0.08*0.20625 ） = 0.0549

    • 留存率是以“登录用户数”为分母，与转移单位cookie不相似，分析单位和转移单位不相似，所以分析估计值与经验变异不相似

  • Std dev = sqrt（0.1093125 * （1-0.1093125） / 5000*0.08） = 0.0156

    • 净转化率是以cookie数量作为分母，也是转移的单位。转移单位等于分析单位，标明分析估计值与经验变异类似

Sizing

规模

Number of Samples vs. Power
样本数量和功效

在分析阶段不准备使用使用 Bonferroni 校正，因为本实验中总转化率和净转化率是相关联的，使用 Bonferroni 校正会使试验结果过于保守。使用的统计显著性水平为 0.05，统计功效设置为 0.80。使用在线计算器计算样本容量

  • Gross conversion（总转化率）
    Minimum detectable effect:1% 样本数量：25835
    总页面浏览量：25835/0.08*2 =645875
  • Retention（留存率）
    Minimum detectable effect:1% 样本数量：39115
    总页面浏览量：39115/0.20625/0.08*2 =4741212
  • Net conversion（净转化率）
    Minimum detectable effect:0.75% 样本数量：27413
    总页面浏览量：27413/0.08*2 =685325

    • 最后比较三种评价度量需要的样本容量，净转化率需要的页面浏览量685325；

Duration vs. Exposure
试验持续时间以及试验实施时间

从实验周期上来说，每天页面总流量是4万，需要的总流量是68.5万，分别计算需要的时间，50%流量需要35天，100%流量需要18天；综合考虑风险容忍度和试验周期，我选择了曝光80%的流量，持续时间22天的方案；

  • 本试验只是询问用户每周能投入多少时间学习，没有不良影响
  • 收集的数据，不涉及隐私，不是敏感数据
  • 对网站来说，没有安全隐患

Sanity Checks

合理性检查

对于每个不变指标，对你在95%置信区间下期望观察到的值、实际观察的值及指标是否通过合理性检查给出结论。对于任何未通过的合理性检查，根据每日数据解释你觉得最有可能的原因。在所有合理性检查通过前，不要开始其他分析工作。

  • Number of cookies 置信区间= [0.4988, 0.5012]
    观察值 = 344660 / 690203 = 0.5006
    通过合理性检查
  • Number of clicks 置信区间= [0.4959, 0.5041]
    观察值 = 28378 / 56703 = 0.5005
    通过合理性检查
  • Click-through-probalility 置信区间= [0.0812, 0.0830]
    观察值 = 0.0821824
    通过合理性检查

Result Analysis

结果分析

Effect Size Tests/效应大小检验

对于每个评估指标，对试验和对照组之间的差异给出 95% 置信区间。说明每个指标是否具有统计和实际显著性。

  • Gross conversion（总转化率）

    • 置信区间= [ -0.0291, -0.0120 ]
    置信区间不包括0，具有统计显著性；
    置信区间不包含d_min，具有实际显著性

  • Net conversion（净转化率）

    • 置信区间= [ -0.01160, 0.001857 ]
    置信区间包括0，不具有统计显著性；
    置信区间包含d_min（+/- 0.0075），不具有实际显著性；

Sign Tests/符号检测

对于每个评估指标，使用每日数据进行符号检验，然后报告符号检验的 p 值以及结果是否具有统计显著性

  • Gross conversion（总转化率）

    • 双尾P值0.0026 小于 alpha 水平0.025，具有统计显著性；

  • Net conversion（净转化率）

    • 双尾P值0.6776 大于 alpha 水平0.025，不具有统计显著性；

Summary

总结&建议

不建议启动试验：总转化率具有统计和实际显著性，通过了预期的假设检验，说明这个措施会减少因为没有足够的时间而离开免费试学的学生数量。净转化率的置信区间包含负数，也就是说有很大的概率净转化率会减少，造成商业上的影响。

Follow-Up Experiment

— 后续实验

Experimental overview

实验概述

目前官网上的课程都是英文的，语速特别快，学员一半时间都用在英文课程的理解上，我准备后续改进一下英文视频的翻译版本，或者直接在登陆后课程一开始提醒这个课程需要一定的英语水平。

假设

留存率下降，但是最后的节课率上升

不变度量

  • 用户id的数量
    此试验就是单纯多了一个提醒，不会影响到登录用户数量，因此选择用户id做不变度量

评估度量

  • 留存率
    试验可能会影响最终付费用户数量，一些英文不好的人就离开了，留存率是个很好的评估指标
  • ID
    可以跟踪观察用户最后有没有最后报名学习。
